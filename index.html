HTML Basics

  • HTML stands for HyperText Markup Language.

  • HTML is the standard language for creating web pages.

  • HTML elements are represented by tags.

  • Common tags include <div>, <p>, and <a>.

  • HTML documents are structured with a <header> and a <footer>.

    • CSS Fundamentals

    CSS stands for Cascading Style Sheets.

    It is used to style HTML elements.

    CSS rules consist of a selector and a declaration block.

    Common properties include color, font, padding, and margin.

    CSS can be applied inline, internally, or externally.

    JavaScript Essentials

    JavaScript is a programming language used to create dynamic content.

    It can manipulate HTML and CSS.

    Common JavaScript functions include document.getElementById() and console.log().

    JavaScript can be added to HTML files using the <script> tag.

    It supports event handling, allowing for interactive web pages.

    Responsive Design

    Responsive design ensures that web pages look good on all devices.

    It uses fluid grids, flexible images, and media queries.

    Media queries allow for different styles based on device characteristics.

    Example of a media query:
    @media (max-width: 600px) { body { background-color: lightblue; } }.

    Web Development Tools

    Various tools can assist in web development.

    Common tools include text editors, version control systems, and browser developer tools.

    Popular text editors are VS Code, Sublime Text, and Atom.

    Version control systems like Git are essential for collaboration.

    Browser developer tools help debug and optimize web pages.
